# Basic Network Sniffer (Python)

## 📌 Description
This project is a basic network sniffer built using Python and Scapy. It captures and analyzes network packets in real-time.

## ⚙️ Features
- Captures live network traffic
- Displays IP addresses
- Detects TCP/UDP protocols
- Shows ports and payload

## 🛠️ Requirements
- Python 3
- Scapy library

## 🚀 Installation
pip install scapy

## ▶️ Usage
Run:
python sniffer.py

## ⚠️ Note
Run as Administrator for proper packet capture.
